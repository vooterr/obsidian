# Что это
- CPython - реализация python на языку C, его интерпретатор
	- Считается одной из самых популярных и самой эталонной версией интерпретатора
## Чем отличается от питона
- Python - это *язык программирования*, а *CPython* - его *интерпретатор*
	- На самом деле отличие колоссальное
	- Если язык - это набор правил и синтаксиса, то интерпретатор - это программа, которая исполняет код на этом языке
	- Можно это сравнить как язык *C* и *gcc*
- **Вкратце, на Python вы пишете код, а CPython - отвечает за исполнение этого кода**
