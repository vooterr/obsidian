# Что это
- CPython - реализация python на языку C, его интерпретатор
	- Считается одной из самых популярных и самой эталонной версией интерпретатора
## Чем отличается от питона
- Python - это *язык программирования*, а *CPython* - его *интерпретатор*
	- На самом деле отличие колоссальное
	- Если язык - это набор правил и синтаксиса, то интерпретатор - это программа, которая исполняет код на этом языке
	- Можно это сравнить как язык *C* и *gcc*
- **Вкратце, на Python вы пишете код, а CPython - отвечает за исполнение этого кода**

## Как компилируется
- При компиляции, код проходит через три этапа
	1. Инициализация интерпретатора
	2. Компиляция исходного кода в байт-код
	3. Исполнение байт-кода
###  1. Инициализация интерпретатора
- На этом этапе CPython готовит все необходимое: 
	- Выделяет основные структуры данных ( int, str )
	- Загружает модули
	- Настраивает система импорта 
### 2. Компиляция исходного кода в байт-код
- Несмотря на на то что, CPython называют интерпретатором, все не так просто
	- Вначале код разбирается лексером на токены
	- После чего токены парсятся и строится синтаксическое дерево
	- Впоследствии дерево переводится в байт-код (именно байт, не машинный)
- Результатом компиляции будет объект кода (code object)
	- Содержащий байт-код, модули и функции
- Обычно код сохраняется с расширением .pyc в pycache ( чтобы не строить код по новой )
### 3. Исполнение байт-кода
-  Байт-код передается на исполнение виртуальной машине CPython 
- Машина читает данный код и исполняет его в код на уровне C++ кода
	- Для этого используется стековая виртуальная машина,
