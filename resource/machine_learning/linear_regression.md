# Определение
- Линейная регрессия - один из простейших алгоритмов машинного обучения, которые заключается в том, чтобы описать зависимость целевой переменной от признаков в виде линейной функции
	- В простом виде выглядит как линия, пересекающее N-мерное пространство 
	- Представим как перемножение f(x) = <x, w>, где x - признаки, а w - веса признаков
### Метод наименьших квадратов, функция потерь
- Метод наименьших квадратов - это метод при котором находится оптимальная прямая, для функции линейной регрессии
	- Оптимальная функция - такая, которая минимизирует функцию потерь *среднеквадратичной ошибки* таким образом, что *разница квадратов ожидаемых и реальных значений была бы минимальной*
#### Минусы
- Метод наименьших квадратов не может адекватно воспринимать выбросы в данных, с этой задаче возможно лучше бы справилась метод наименьших модулей, в которой фигурируют модули.
	- Но функция модуля не дифференцируема в нуле, поэтому становится проблематично минимизировать ошибку модуля. 
- Тогда исходя из теоремы [[gause_markov_term|Гауса Маркова]]
### Принципы работы линейной регрессии
- Существуют два основных способа обучения линейной регрессии
	- *Прямое (нормальное) уравнение* в аналитическом виде $\omega = (X^{T}X)^{-1}X^{T}y$, где $\omega$ - вектор весов, включая смещение $b$ . Однако данный способ требует *высокой вычислительной мощности*
	- *Итеративная оптимизация с постепенным снижением ошибки* модели на основе градиентного спуска и его разновидностей. Данный способ чаще всего используется *на практике*
#### Поиск решение через нормальное уравнение
- Пусть у нас есть $X \in \mathbb{R}^{n\times m}, y \in \mathbb{R}^{n \times 1}, w \in \mathbb{R}^{m\times 1}$, необходимо определить оптимальные веса для модели $a(X, w)$, в которой функционалом качества будет средне квадратичная ошибка:
	- $L(w) = ||Xw - y||^2 \to \min$ 
	- Для этого необходимо найти производную и градиент функционала:
		- $dL(w) = d(||Xw - y||^{2)}$
	- Решим задачу аналитически: 
		- $$\begin{aligned}&dL(w) = d(||Xw - y||^{2}) = (Xw - y)^{T} \cdot (Xw - y) = \\& = d(w^{T}X^{T}Xw - y^{T}Xw - w^{T}X^{T}y + y^{T}y) =\\&= d(w^{T}X^{T}Xw) - 2d(y^{T}Xw) = \\ &=(dw^{T}X^{T}Xw + w^{T}X^{T}Xdw) -2y^{T}Xdw =\\&= 2(w^{T}X^{T}Xdw - 2y^{T}Xdw) \end{aligned}$$
		- Приведем функцию к нормальному для градиент ($\nabla L(w) dw$):
			- $$\begin{aligned}&dL(w) = 2(w^{T}X^{T}Xdw) - 2(y^{T}Xdw)=\\ &= 2(w^{T}X^{T}X - y^{T}X)dw \rightarrow \\ &\rightarrow\nabla L(w) = 2(w^{T}X^{T}X - y^{T}X)\end{aligned}$$
		- Приравняем градиент к нулю и найдем минимум функции
			- $$\begin{aligned} &\nabla L(w) = 2(w^{T}X^{T}X - y^{T}X) = 0 \rightarrow \\ &
				\rightarrow w^{T}X^{T}X = y^{T}X \xrightarrow{T}\\ &\xrightarrow{T} X^{T}Xw = X^{T}y \rightarrow \\ &\rightarrow w = (X^{T}X)^{-1} X^{T}y \end{aligned}$$
#### Поиск решения при помощи численных методов
[[gradiet_descent|Градиентный спуск]]
- Изначально устанавливаются *нулевые значения* для *весов* , *смещения* и их *градиентов*
- На основе установленных значений делается прогноз, в какую сторону следует сместить веса
- На основе полученного прогноза пересчитываются значения весов и смещения, а также снижение их градиентов (разница предыдущего и текущего шага)
- Шаги 2 и 3 повторяются, пока снижение градиентов не станет меньше порогового значения
- Итогом станет линейная комбинация полученных весов, смещение признаков на тестовой выборке
## Полиномиальная регрессия
- Линейную модель можно использовать там, где зависимость нелинейна
- Путем перемножения каждого признака друг на друга, и на самого себя
- Такой подход называет *полиномиальной регрессией*, в ScikitLearn имеется класс PolynomialFeatures, для полиномов степени n
- Например для признаков $a, b$ со степенью 3 получатся такие новые признаки
	- $a^{3}, a^{2}b, ab^{2}, b^{3}, ab, a^{2}, b^{2}$  
- **Полученная модель получится также линейной, однако на плоскости она будет казаться кривой**, так как полученная плоскость признаков будет соответствовать сложной кривой в исходном пространстве
### Минусы
- Нужно внимательно следить за получающейся функцией
	- При достаточно большой степени полиномов, может получатся переобученной
	- При меньшем количестве степеней, может оказать недообученной 
### Регуляризации линейной регрессии 
- Если в полиномиальной регрессии регуляризация позволяет уменьшить кривизну линии через снижение количества полиномиальных степеней, то в случае с линейной регрессией регуляризация заключается в изменении наклона линии путем ограничения весов модели, обменивая более высокое смещение на низкую дисперсию
- Если модель *мултиколлинеарна* -> Ridge
- Если необходимо *отобрать наиболее важные признаки* -> Lasso
- Если *необходимо и то и то* -> ElasticNet
#### Ridge
$$J+= \alpha \sum\limits^{n}_{i=1}w^{2}_{i}$$
- Гребневая регрессия или регуляризация Тихонова применяет в случае *мультиколлинеарности через добавление L2-регуляризации к функции потерь*
- Во время обучения силнее всего занижает веса для признаков с высокой корреляцией
	- Их значения будут приближаться к нулю, но никогда его не достигнут. 
	- Лучше применять ее после *стандартизации признаков*
- Lasso тоже помогает при мультиколлинеарности (просто выбирает один признак из группы коррелирующих), но Ridge делает это «мягче», сохраняя все признаки.
#### Lasso
$$J+=\alpha \sum\limits^{n}_{i=1}|w_{i}|$$
- **Least absolute shrinkage & selection operator(Lasso)**
- Обычно используется для отбора признаков через добавление *L1-регуляризации* к функции потерь во время обучения
	- Проще говорят, Lasso функция стремится *уменьшить число параметров модели, путем зануления весов для неинформативных и избыточных признаков*
	- Что даст на выходе разряженную модель и количество признаков
#### ElasticNet
- Эластичная сеть представляет между собой *L1 и L2-регуляризации через отношение их смеси r*
	- Это может принести пользу в стациях когда данные необходимо *одновременно* выполнять *отбор признаков и бороться с мультиколлинеарностью*
	- Для оперирования смеси ridge и lasso используется `l1_ratio`
## Геометрическая интерпретация 
- Линейная регрессия по своей сути является гиперпространством размерности n, которая *проецирует на себя признаки* объекта, *задавая каждому признаку какой то определенный базис линейного пространства*
