## Метрики
- Метрики необходимы для определения качества моделей, от выбора метрики часто зависит работа всего алгоритма
	- На одной метрике модель может выдавать хорошие результаты, когда на другой плохие
	- Поэтому очень важно подобрать хорошие метрики для своей задачи
- Часто их еще называют **функционалом ошибки**
### Метрики для задач
- Для каждой задаче обычно придумывают разные метрики, например для задачи регрессии метрики будут сильно от метрик для задач классификации
