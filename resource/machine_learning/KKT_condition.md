# Условия Каруша–Куна–Такера (KKT)

## 📝 Основное содержание

- Необходимые условия оптимальности для задач оптимизации **с ограничениями-неравенствами**.
- Обобщение метода множителей Лагранжа на случай неравенств.
- В SVM используются для перехода к двойственной задаче и классификации опорных объектов.

## Зачем нужны KKT — интуиция

Представь простую задачу: минимизировать $f(x)$ без ограничений. Решение — там, где $\nabla f = 0$, градиент обнуляется.

Теперь добавим ограничение: $g(x) \leq 0$. Возникают два случая:

- **Ограничение неактивно** ($g(x^*) < 0$): оптимум лежит внутри допустимой области, ограничение не «давит» — решение то же, что и без него. Множитель $\lambda = 0$.
- **Ограничение активно** ($g(x^*) = 0$): оптимум упёрся в границу. Градиент $f$ теперь уравновешивается градиентом $g$. Множитель $\lambda > 0$.

KKT формализуют эту логику для произвольного числа ограничений.

---

## Постановка задачи

Общая задача оптимизации с ограничениями:

$$\begin{cases} f(x) \to \min_x \[4pt] g_i(x) \leq 0, \quad i = 1, \dots, m \[4pt] h_j(x) = 0, \quad j = 1, \dots, k \end{cases}$$

Для каждого ограничения вводятся **множители Лагранжа**:

- $\lambda_i \geq 0$ — для ограничений-неравенств $g_i$
- $\mu_j \in \mathbb{R}$ — для ограничений-равенств $h_j$

**Функция Лагранжа:** $$\mathscr{L}(x, \lambda, \mu) = f(x) + \sum_{i=1}^{m}\lambda_i g_i(x) + \sum_{j=1}^{k}\mu_j h_j(x)$$

## Условия KKT

Если $x^*$ — локальный минимум и выполнены условия регулярности (например, ограничения линейны), то в точке $x^{*}$ выполняются **четыре группы условий**:

### 1. Стационарность

$$\nabla_{x} \cal{L}(x^*, \lambda, \mu) = 0$$ $$\nabla f(x^*) + \sum_{i}\lambda_i \nabla g_i(x^*) + \sum_{j}\mu_j \nabla h_j(x^*) = 0$$

Градиент целевой функции компенсируется градиентами ограничений. В оптимуме «двигаться» некуда — любое направление улучшения нарушает хотя бы одно ограничение.

### 2. Допустимость (primal feasibility)

$$g_i(x^*) \leq 0, \quad h_j(x^*) = 0$$

Решение обязано лежать в допустимой области — выполнять все ограничения задачи.

### 3. Двойственная допустимость

$$\lambda_i \geq 0$$

Множители для ограничений-неравенств не могут быть отрицательными. Это следует из геометрии: в точке оптимума активное ограничение «толкает» решение внутрь, а не наружу.

### 4. Условия дополняющей нежёсткости (complementary slackness)

$$\lambda_i \cdot g_i(x^*) = 0, \quad i = 1, \dots, m$$

Это ключевое условие. Для каждого $i$ **ровно одно из двух**:

- $\lambda_i = 0$ — ограничение неактивно, множитель нулевой (ограничение «не давит»)
- $g_i(x^*) = 0$ — ограничение активно, множитель может быть ненулевым

Они не могут оба быть ненулевыми одновременно. Именно это условие позволяет классифицировать объекты в SVM.

---

## Геометрический смысл

В точке минимума на границе допустимой области градиент $\nabla f$ должен быть направлен **против** нормали к границе (иначе можно было бы сдвинуться вдоль границы и уменьшить $f$):

$$\nabla f(x^{*}) = -\sum\limits_{i} \lambda_{i} \nabla g_{i}(x^{*}), \quad \lambda_{i} \geq 0$$

Геометрически: $\nabla f$ лежит в конусе, натянутом на нормали к активным ограничениям. Если $\nabla f$ «вырывается» из этого конуса — оптимум не найден.

## Применение в SVM

В задаче SVM с мягким зазором: $$\frac{1}{2}|w|^2 + C\sum_i \xi_i \to \min, \qquad M_i(w,w_0) \geq 1 - \xi_i, \quad \xi_i \geq 0$$

Ограничения-неравенства: $g_i = 1 - \xi_i - M_i \leq 0$ и $-\xi_i \leq 0$.

Множители: $\lambda_i \geq 0$ для первого ограничения, $\eta_i \geq 0$ для второго.

Условие дополняющей нежёсткости даёт: $$\lambda_i(M_i - 1 + \xi_i) = 0 \quad \text{и} \quad \eta_i \xi_i = 0$$

Именно отсюда берётся **типизация объектов**:

|$\lambda_i$|$\xi_i$|$M_i$|Тип объекта|
|---|---|---|---|
|$= 0$|$= 0$|$> 1$|Периферийный — вне зазора, не влияет на $w$|
|$(0, C)$|$= 0$|$= 1$|Опорный — ровно на границе зазора|
|$= C$|$> 0$|$< 1$|Опорный нарушитель — внутри зазора или за ним|

> **Вывод:** $w = \sum_i \lambda_i y_i x_i$ — вектор весов выражается **только через опорные объекты** (у периферийных $\lambda_i = 0$ и они в сумму не входят).

## Достаточность KKT

KKT — необходимые условия. Для выпуклых задач (а SVM — выпуклая) они становятся **необходимыми и достаточными**: любая точка, удовлетворяющая KKT, является глобальным минимумом.

Это и делает KKT центральным инструментом анализа SVM — у задачи нет локальных минимумов, в которых можно «застрять».
