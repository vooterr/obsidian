# GLM — Обобщённая линейная модель

## 📝 Основное содержание

### Мотивация: почему нужно обобщение

Обычная линейная регрессия предполагает, что $y_i = x_i^\top \beta + \varepsilon_i$, где $\varepsilon_i \sim \mathcal{N}(0, \sigma^2)$. Это накладывает три ограничения: отклик $y$ может принимать любые вещественные значения, среднее линейно зависит от $x$, и дисперсия постоянна. Как только мы выходим за эти рамки, модель начинает давать физически бессмысленные результаты.

Например, если отклик — вероятность клика ($y \in [0, 1]$), линейная регрессия может предсказать $1.7$ или $-0.3$. Если отклик — число звонков за день ($y \in {0, 1, 2, \ldots}$), предсказание может оказаться отрицательным. GLM решает эту проблему, не отказываясь от линейности, а добавляя «переходник» между линейной комбинацией признаков и областью допустимых значений отклика.

---

### Три компонента GLM

GLM — это не одна модель, а **рецепт** для построения модели, состоящий из трёх выборов. Каждый выбор отвечает за свою часть задачи.

#### Компонент 1: распределение отклика

Предполагается, что каждое наблюдение $y_i$ подчиняется некоторому распределению из **экспоненциального семейства** (подробнее ниже). Распределение выбирается исходя из природы данных: бинарные метки — распределение Бернулли, счётные данные — Пуассон, непрерывные положительные — Гамма, обычные непрерывные — нормальное.

#### Компонент 2: линейный предиктор

Линейная комбинация признаков остаётся неизменной:

$$\eta_i = x_i^\top \beta$$

Линейный предиктор $\eta_i$ всегда живёт на всей числовой прямой $(-\infty, +\infty)$, независимо от природы отклика. Это та часть модели, которую мы умеем хорошо оценивать — линейность даёт удобную математику и интерпретируемость коэффициентов.

#### Компонент 3: функция связи

Это ключевой элемент GLM. **Функция связи $g$** — монотонная дифференцируемая функция, которая связывает ожидаемое значение отклика $\mu_i = \mathbb{E}[y_i \mid x_i]$ с линейным предиктором:

$$g(\mu_i) = \eta_i = x_i^\top \beta$$

Или в обратную сторону: $\mu_i = g^{-1}(x_i^\top \beta)$.

Задача функции связи — «растянуть» ограниченную область допустимых значений $\mu$ на всю числовую прямую, где живёт $\eta$. Именно поэтому модель не даёт бессмысленных предсказаний: линейная комбинация может быть любым числом, но после применения $g^{-1}$ результат всегда попадает в допустимую область.

Для каждого распределения существует **каноническая функция связи** — «естественная» для данного распределения, при которой математика оптимизации становится максимально простой. Для Бернулли это логит $g(\mu) = \log\frac{\mu}{1-\mu}$, для Пуассона — логарифм $g(\mu) = \log\mu$, для нормального — тождественная $g(\mu) = \mu$.

---

### Экспоненциальное семейство

Это класс распределений, объединённых общей структурой. Распределение принадлежит экспоненциальному семейству, если его плотность можно записать в виде:

$$p(y \mid \theta, \phi) = \exp!\left(\frac{y\theta - b(\theta)}{a(\phi)} + c(y, \phi)\right)$$

Каждый элемент этой формулы имеет конкретный смысл.

**$\theta$** — **канонический параметр** (натуральная параметризация распределения). Это перепараметризация привычного параметра в такой форме, при которой логправдоподобие принимает наиболее удобный вид. Для Бернулли $\theta = \log(\mu/(1-\mu))$ — тот самый логит. Для Пуассона $\theta = \log\mu$ — логарифм среднего.

**$b(\theta)$** — **функция кумулянтов** (нормировочная функция). Она гарантирует, что вероятности суммируются в 1. Но главное её свойство — её производные задают моменты распределения:

$$b'(\theta) = \mathbb{E}[y] = \mu \qquad b''(\theta) = \frac{\text{Var}(y)}{a(\phi)}$$

Это означает, что зная $b(\theta)$, мы автоматически знаем, как дисперсия зависит от среднего. Именно через $b''$ дисперсия войдёт в веса $W$ при оптимизации.

**$a(\phi)$** — **дисперсионная функция**, масштабирующий множитель. Для нормального распределения $a(\phi) = \sigma^2$, для Бернулли и Пуассона $a(\phi) = 1$ — их дисперсия полностью определяется средним.

**$c(y, \phi)$** — нормировочный член, зависящий от данных, но не от $\theta$. При оптимизации по $\beta$ он выпадает как константа.

Чтобы убедиться, что привычное распределение — частный случай, достаточно переписать его логплотность в данной форме. Для Пуассона: $\log p(y|\lambda) = y\log\lambda - \lambda - \log(y!)$. Полагая $\theta = \log\lambda$, получаем $b(\theta) = e^\theta$, $a(\phi) = 1$, $c(y,\phi) = -\log(y!)$. Тогда $b'(\theta) = e^\theta = \lambda = \mu$ и $b''(\theta) = e^\theta = \mu$ — дисперсия Пуассона равна среднему, как и должно быть.

---

### Метод максимального правдоподобия

Параметры $\beta$ оцениваются максимизацией логарифма правдоподобия. Для экспоненциального семейства:

$$\ell(\beta) = \sum_{i=1}^{n} \frac{y_i \theta_i - b(\theta_i)}{a(\phi)} + \text{const}$$

Поскольку $\theta_i$ зависит от $\beta$ через цепочку $\beta \to \eta_i \to \mu_i \to \theta_i$, нахождение максимума требует итеративного алгоритма. Необходимое условие максимума: $\nabla_\beta \ell(\beta) = 0$.

#### Вычисление градиента

Применяя цепное правило по цепочке зависимостей, градиент принимает вид:

$$\nabla_\beta \ell = X^\top (y - \mu)$$

Это ключевая формула. Вектор $(y - \mu)$ — остатки между наблюдёнными значениями и предсказанными средними. Умножение на $X^\top$ — проецирование остатков в пространство параметров. Структура точно такая же, как в МНК, — только $\mu$ теперь нелинейно зависит от $\beta$.

#### Вычисление гессиана

Вторая производная логправдоподобия:

$$H = -X^\top W X$$

где $W = \text{diag}(w_1, \ldots, w_n)$ — диагональная матрица весов:

$$w_i = \frac{1}{a(\phi)} \cdot b''(\theta_i) \cdot \left(\frac{d\mu_i}{d\eta_i}\right)^2$$

Первый множитель $b''(\theta_i)$ — дисперсия отклика из свойств экспоненциального семейства. Второй множитель $(d\mu_i/d\eta_i)^2$ — чувствительность среднего к изменению линейного предиктора. При канонической функции связи они сокращаются, и $w_i = 1/\text{Var}(y_i)$: наблюдения с большей дисперсией получают меньший вес.

---

### Алгоритм IRLS

Применяем к задаче максимизации $\ell(\beta)$ метод Ньютона–Рафсона. Шаг обновления:

$$\beta_{t+1} = \beta_t - H^{-1} \nabla_\beta \ell = \beta_t + (X^\top W X)^{-1} X^\top (y - \mu)$$

Введём вектор **скорректированного отклика**:

$$z = \eta_t + W^{-1}(y - \mu_t) = X\beta_t + W^{-1}(y - \mu_t)$$

Тогда шаг Ньютона переписывается как:

$$\boxed{\beta_{t+1} = (X^\top W X)^{-1} X^\top W z}$$

Это в точности формула **взвешенного МНК** с матрицей весов $W$ и откликом $z$. Каждая итерация метода Ньютона для GLM — это решение одной задачи взвешенной линейной регрессии. Именно поэтому алгоритм называется **IRLS** (Iteratively Reweighted Least Squares — итерационно перевзвешенный МНК).

На каждой итерации выполняется следующая последовательность шагов. Сначала по текущим $\beta_t$ вычисляются предсказания $\eta_t = X\beta_t$ и средние $\mu_t = g^{-1}(\eta_t)$. Затем пересчитываются веса $W$ и скорректированный отклик $z$. После этого решается взвешенная МНК задача для нахождения $\beta_{t+1}$. Повторяется до сходимости $|\nabla\ell(\beta)| < \varepsilon$.

#### Веса для конкретных распределений

Для **логистической регрессии** (Бернулли, логит-связь) веса $w_i = \mu_i(1 - \mu_i)$. Наблюдения вблизи $\mu_i = 0.5$ получают максимальный вес — именно там неопределённость максимальна. Наблюдения с уверенными предсказаниями ($\mu_i \approx 0$ или $\mu_i \approx 1$) получают малый вес.

Для **пуассоновской регрессии** (Пуассон, лог-связь) веса $w_i = \mu_i$. Чем больше ожидаемое число событий, тем больший вес — потому что дисперсия Пуассона равна среднему, и при большем среднем данные несут больше информации о параметрах.

---

### Гарантии сходимости

Для GLM с канонической функцией связи логправдоподобие **строго вогнуто** по $\beta$ — это гарантируется тем, что $X^\top W X$ положительно определена при $w_i > 0$ и $X$ полного ранга. Это означает единственность максимума и глобальную сходимость IRLS из любого начального приближения. На практике алгоритм сходится за 5–10 итераций, унаследовав квадратичную скорость метода Ньютона–Рафсона.
