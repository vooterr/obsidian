## Определение
- *Bias-variance tradeoff* - это баланс между систематической ошибкой модели и ее чувствительностью к данным, оптимум достигается при минимуме ошибок на новых данных
### Идея
- *Bias*(смещение) - модель слишком простая и не улавливает закономерностей
- *Variance*(разброс) - модель слишком сложная, подстраивается под шум
- Хороший результат достигается при одинаково низких данных параметрах
	- Иногда при сильном занижении одного параметра, второй будет сильно расти
- $$y = (f(x) + \epsilon), \mathbb{E}[\epsilon] = 0, Ver(\epsilon) = \sigma^{2} $$
- $$\mathbb{E}[(y - f(x))^{2}] = (\mathbb{E}[f(x)] - f(x))^{2} + Var(f(x))+ \sigma^{2}, \text{где } \mathbb{E}(f(x) - f(x))^{2} \text{ - Bias, } \\ Var(f(x)) \text{ - Variance, } \sigma^{2} \text{ - небольшой шум}$$
- Простые модели -> высокий bias, низкий variance
- Сложные модели -> низки bias, но высокий variance
