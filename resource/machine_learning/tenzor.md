# Определение
- **Тензор** - это математический объект, который является обобщением чисел, векторов и матриц на более высоком порядке ( размерности ). Его можно представить как многомерный массив данных
	- В зависимости от ранга, тензоры классифицируется следующим образом
		1. Порядок = 0: Скаляр или **просто число**
		2. Порядок = 1: Вектор или **упорядоченный набор чисел**
		3. Порядок = 2: Матрица или двумерный массив чисел
		4. Порядок = N, где N > 1: многомерный массив размерности N или **тензор высокого порядка**
- **Математическое определение тензора** - это массив чисел, расположенных на регулярной сетке с переменным числом осей. Тензор является обобщением понятий, используемых в линейной алгебре, на более высокий порядок
### Для чего нужен тензор
- Тензор необходим тогда, когда невозможно представить данные матрицей или вектором чисел, необходима размерность намного больше. 
	- В машинном обучении они служат представлением данных в нейронных сетях и алгоритмах
### Обозначение
- Обозначаются как и матрица, большой буквой, например $A$,
- Элементы обозначаются как и в матрице при помощи индексов:
	- $A_{i, j, k}$ - элемент тензора с *координатами i, j, k*
## Свойства тензора
### Broadcasting
- Broadcasting( транслирование ) - специфичный метод сложения тензоров разной размерности
	- *Меньший тензор неявно копируется, дублируется для каждой строки матрицы*, чтобы размерности совпали, что позволяет выполнить сложение без физического создания копии данных 
### Дифференцирование
- Можно вычислять градиент для значения z по отношению к тензору $\mathbb{X}$
- Обозначаться градиент будет $\nabla_{\mathbb{X}}z$. Если X - тензор, то градиент также будет тензором, то же формы, где элемент с индексом i равен частно производной z по соответствующему элементу X
