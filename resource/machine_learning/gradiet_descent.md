## Суть градиентного спуска
**Градиентный спуск** — это итеративный алгоритм оптимизации. Его главная цель — найти такие параметры модели (веса), при которых функция потерь (ошибка) достигает своего минимума.
**Интуиция:** Представь, что ты стоишь с завязанными глазами на склоне горы и хочешь спуститься в самую низкую точку. Ты щупаешь землю ногой, определяешь, в какую сторону наклон самый крутой, и делаешь шаг в противоположном направлении.
**Математика базового шага:**
Формула обновления весов на каждой итерации $t$ выглядит так:
$$w_{t+1} = w_t - \eta \cdot \nabla L(w_t)$$

Где:
- **$w$** — параметры (веса) модели.
- **$L(w)$** — функция потерь (Loss function), которую мы минимизируем.
- **$\nabla L(w_t)$** — градиент функции потерь по весам. Это вектор, который показывает направление **наискорейшего возрастания** функции.
- **$\eta$** (Learning Rate, шаг обучения) — гиперпараметр, определяющий размер шага. Мы ставим минус перед градиентом, потому что нам нужно идти в сторону убывания (антиградиент).

## 2. Эвристики и практические правила
Даже базовая версия алгоритма требует правильной настройки, иначе модель либо не сойдется, либо будет учиться вечность. Вот основные эвристики:
- **Масштабирование признаков (Feature Scaling / Normalization):**
    - _Проблема:_ Если один признак измеряется в тысячах, а другой в долях, линии уровня функции потерь вытягиваются в узкие эллипсы. Градиентный спуск начинает «осциллировать» (прыгать из стороны в сторону) и очень медленно движется к центру.
    - _Решение:_ Нормализация или стандартизация (например, вычитание среднего и деление на стандартное отклонение). Линии уровня становятся круглыми, и спуск идет прямо к цели.
- **Подбор Learning Rate ($\eta$):**
    - _Слишком большой $\eta$:_ Алгоритм «перешагнет» минимум и может вообще разойтись (ошибка начнет расти).
    - _Слишком маленький $\eta$:_ Очень медленное обучение и риск застрять в локальном минимуме.
    - _Эвристика:_ Начинать с небольших значений (например, 0.01, 0.001) и использовать расписание (Learning Rate Decay) — уменьшать шаг со временем, когда подбираемся ближе к минимуму.
- **Инициализация весов:**
    - Нельзя инициализировать все веса нулями в нейросетях (возникнет симметрия, все нейроны будут учиться одинаково).
    - _Эвристика:_ Инициализировать веса случайными малыми числами (например, из нормального распределения).
- **Критерии остановки (Early Stopping):**
    - Как понять, что пора остановиться? Алгоритм прерывают, если:
        1. Достигнуто максимальное число эпох.
        2. Норма вектора градиента стала близка к нулю (мы на дне).
        3. Ошибка на валидационной выборке перестала падать (чтобы избежать переобучения).
> **Связь с продвинутыми методами:** Базовый GD (Batch GD) считает градиент по всему датасету сразу, что очень долго. Поэтому на практике используют Stochastic Gradient Descent (SGD) — по одному объекту, или Mini-batch GD — по пачкам объектов. А чтобы бороться с застреванием в локальных минимумах и осцилляциями, применяют методы с инерцией и адаптивным шагом (Momentum, RMSprop, Adam).
```python
import numpy as np

def gradient_descent(X, y, learning_rate=0.01, epochs=1000, tolerance=1e-6):
    """
    X: матрица признаков (n_samples, n_features)
    y: вектор целевой переменной (n_samples,)
    """
    n_samples, n_features = X.shape
    
    # 1. Инициализация весов (предполагается, что в X уже добавлен столбец единиц для bias)
    weights = np.zeros(n_features) 
    
    # Для отслеживания истории ошибки
    loss_history = []
    
    for i in range(epochs):
        # 2. Предсказание модели (Forward pass)
        predictions = np.dot(X, weights)
        
        # 3. Вычисление ошибки
        error = predictions - y
        
        # Функция потерь (MSE) для истории
        mse = np.mean(error**2)
        loss_history.append(mse)
        
        # 4. Вычисление градиента (матрично)
        # Формула градиента для MSE: (2/n) * X.T * (X*w - y)
        gradient = (2 / n_samples) * np.dot(X.T, error)
        
        # 5. Обновление весов (Шаг спуска)
        weights -= learning_rate * gradient
        
        # 6. Критерий ранней остановки (эвристика)
        if np.linalg.norm(gradient) < tolerance:
            print(f"Сходимость достигнута на эпохе {i}")
            break
            
    return weights, loss_history
```
