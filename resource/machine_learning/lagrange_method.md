# lagrange_method

## 📝 Основное содержание
- В данной статьей разбирается метод Лагранжа, которой способен находит точки локального минимума при некотором условии

## 🔗 Связи

## Зачем нужен
- Представим, что нам необходимо найти локальный минимум в точке $f(x, y) = x^{2} + y^{2}$ 
	- Для данной задачи ответ очевиден: $(0, 0)$
- Но если представить, что в нашей функции теперь есть ограничения, для всех точек, которые лежать в некотором пространстве $S$, которое задается функцией $g(x, y) = x + y - 1$
	- Получается уже не такая элементарная задача
- Именно здесь на сцену выходит Лагранж. Его ключевая идея звучит так: 
	- В точке оптимума градиент целевой функции $\nabla f$ и градиент ограничения $\nabla g$ **параллельны** — то есть один является масштабированной версией другого.
## Алгоритм
### Условия
- Для выполнения метода Лагранжа необходимы некоторые условия
	1. Функция $f$ - непрерывно дифференцируема 
	2. Функция $g$  непрерывно дифференцируема, с частными производными, не равными нулю одновременно, то есть уравнение $g(x, y)$ задаёт гладкую кривую $S$ из обыкновенных точек на плоскости $(x, y)$
	3. Кривая $S$ проходит через точки, в которых градиент $f$ обращается в 0
### Работа
- Из условия параллельности градиентов: 
	- $\nabla f = \lambda \nabla g$
	- Где $\lambda$ - множитель Лагранжа
- Вместо того чтобы решать это уравнение напрямую, Лагранж предложил элегантный трюк — ввести **функцию Лагранжа** (лагранжиан):
	- $\cal{L}(x, y, \lambda) = f(x, y) - \lambda g(x, y)$ 
- Тогда условием оптимума - это просто приравнивание всех частных производных Лагранжиана к нулю:
	- $$\frac{\delta \cal{L}}{\delta x} = 0, \frac{\delta \cal{L}}{\delta y} = 0, \frac{\delta \cal{L}}{\delta \lambda} = 0$$
	- Где третье условие дает нам $g(x, y) = 0$
