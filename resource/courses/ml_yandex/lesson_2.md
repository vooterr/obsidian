# Линейные методы классификации и регрессии: метод стохастического анализа
## Метод стохастического градиента
### Обучении регрессии - это оптимизации
- Обучающая выборка: $X^{l} = (x_{i}, y_{i})^{l}_{i=1}, x_{i} \in \mathbb{R}^{n},y_{i}\in \mathbb{R}$
- Модель регрессии - линейная
	- $a(x, w) = <x, w> = \sum\limits_{j=1}^{n} w_{j}f_{j}(x), w\in \mathbb{R}^{n}$
- Функция потерь - квадратичная:
	- $\mathfrak{L}(a, y) = (a - y)^{2}$
	- Чем меньше квадрат разности, тем меньше мы штрафуем
- Метод обучения - метод наименьших квадратов
	- $Q(w)=\sum\limits_{i=1}^{l}(a(x_{i}, w) - y_{i}) \rightarrow \min_{w}$
### Обучение классификации - тоже оптимизация
- Дискретная задач, а следовательно проблематично применять оптимизации к 
- Модель классификации - линейная
	- $a(x, w) = sign\langle x, w\rangle = sign \sum\limits^{n}_{j=1} w_{j}f_{j}(x)$ 
- Функция потерь - бинарная или ее ароксимация: 
	- $\cal{L}(a, y) = [ay < 0] = [\langle x, w\rangle y < 0] \leq \cal{L}(\langle x, w\rangle y)$ 
- Метод обучения - минимизация эмпирического риска 
	- $Q(w) = \sum\limits^{l}_{i=1}[\langle x_{i}, w \rangle y_{i} < 0] \leq \sum\limits^{l}_{i=1}\cal{L}(\langle x_{i}, w \rangle y_{i}) \rightarrow \min_{w}$
- Проверка по тестовой выборке $X^{k}= (\tilde{x_{j}}, \tilde{y_{i}})^{k}_{i=1}$ 
	- $\bar{Q}(w) = \frac{1}{k} \sum\limits^{k}_{i=1}[\langle \tilde{x_{i}}, {w}\rangle \tilde{y_{i}} < 0]$

## Градиентные методы численной минимизации
- Минимизация эмпирического риска (регрессия, классификация)
	- $Q(w) = \sum\limits^{l}_{i=1}\cal{L}_{i}(w) \rightarrow \min_{w}$
- Градиентный метод основан на вычислении *градиента* и постройка по нему пути к минимуму
	- Градиент вычисляется для каждой переменной по отдельности 
- Численная минимизация методом градиентного спуска
	- $w^{(0)} := \text{Начальное приближение}$
	- $w^{(t+1)} := w^{(t)}-h \dot \nabla Q(w^{t})$, $\nabla Q(w) = {({\frac{\delta Q(w)}{\delta w_{j}}})}^{n}_{j=0}$
	- где h - градиентный шаг, называемый также темпом обучения
		- $w^{(t+1)}:= w^{(t)}-h \sum\limits^{l}_{i=1}\nabla \cal{L}_{i}(w^{(t)})$ 
- Хорошая идея *брать градиент не от всей выборки*, а брать множество от выборки, и вычислять градиент от нее
	- Таким образом за конечное число шагов мы придем к хорошему результату
	- Не теряя при этом много времени вычисляя от всей большой выборки
### Стохастический градиент (SG)
- Методы стохастического градиента - обычный градиентный спуск, но с небольшим улучшение
	- Градиент вычисляется не по всей выборке, а на ее части
	- Из выборки случайным образом берется часть объектов (батч или batch)
- На вход подается *выборка*($X^{l}$, *темп обучения*$(h)$ и *темп забывания*($\lambda$)
- Шаги SG:
	1. Инициализируются веса (рандомно либо по каким нибудь эвристикам)
	2. Инициализируется оценка функционала
		- $Q := \frac{1}{l} \sum\limits^{l}_{i=1}\cal{L}_{i}(w)$ 
	3. Повторять
		1. Выбираем объект $x_{i}$ случайным образом
		2. вычислить потерю $\epsilon_{i}:= \cal{L}_{i}(w)$
		3. Сделать градиентный шаг: $w:= w - h\nabla\cal{L}_i(w)$
		4. Оценить функционал: $Q : = \lambda\epsilon_{i}+(1 - \lambda) Q$
	4. Пока значения функционала или веса w не сойдутся 
- **Берем случайные значения** потому что данные могут зацикливаться и градиент будет крутиться на месте 
#### Проблема оценки функционала
- Очень долго вычислять каждый раз функционал, вплоть до того, что выборка по стохастическому градиенту становится бессмысленна, потому что мы проходимся столько же по данным
- **Как решение** можно использовать приближенную рекуррентную формулу 
- *Среднее арифметическое*: 
	- $\bar{Q}_{m}= \frac{1}{m}\epsilon_{m} + \frac{1}{m}\epsilon_{m-1} + \frac{1}{m} \epsilon_{m-2} + \dots$
	- $\bar{Q}_{m} = \frac{1}{m}\epsilon_{m} + (1 - \frac{1}{m})\bar{Q}_{m-1}$
- *Экспоненциальное скользящее среднее*:
	- $\bar{Q}_{m}= \lambda \epsilon_{m}+ (1 - \lambda) \lambda \epsilon_{m-1} + (1-\lambda)^{2}\lambda \epsilon_{m-2}$ 
	- $\bar{Q}_{m}=\lambda \epsilon_{m} + (1 - \lambda) \bar{Q}_{m-1}$
	- **Чем больше лямбда, тем силнее геометрическая прогрессия убывает в прошлое** ( старые данные забываются)
### Стохастический средний градиент (SAG)
- Ведет более целенаправленные шаги в сторону истинного градиента
	- Есть еще много других вариантов реализации скольжения в модели. 
	- Можно использовать ту же самую экспоненциальную функцию
- Алгоритм:
- Первые шаги такие же, как и в *стохастическом градиенте*, добавляется еще один шаг:
	- **Инициализация градиента**: $G_{i}:= \nabla \cal{L}_{i}(w), i=1, \dots, l$
- Повторять
	1. Выбрать объект из $X^{l}$ случайным образом
	2. Вычислить потерю: $\epsilon_{i}:= \cal{L}_{i}(w)$
	3. **Вычислить градиент** $G_{i}:= \nabla \cal{L}_{i}(w)$ 
	4. Сделать градиентный шаг: $w := w - h \frac{1}{l} \sum\limits^{l}_{i=1}G_{i}$ 
	5. Оценить функционал: $\bar{Q} := \lambda \epsilon_{i} + (1 - \lambda) \bar{Q}$ 
- Пока функционал или веса на сойдутся 
### Инициализация весов и порядок объектов 
- Инициализация нулями
	- $w_{j} = 0 \text{ для всех } j = 0, \dots, n$ 
- Небольшие случайные значения
	- $w_{j}= random (-\frac{1}{2n}; \frac{1}{2n})$ 
- Эвристика:
	- $w_{j} = \frac{\langle y, f_{i} \rangle}{\langle f_{i}, f_{i} \rangle}$ , $f_{i}= (f_{j}(_{i}))^{l}_{i=1}$ 
	- **Оптимальная, если**
		- Функция потерь квадратичная и
		- Признаки некоррелированные $\langle f_{j}, f_{k}\rangle = 0, j\neq k$
- *Мультистарт*:
	- Многократные запуски из разных случайных начальных приближений и выбор лучшего решения
