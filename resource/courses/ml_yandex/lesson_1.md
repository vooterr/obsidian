# Основные понятия и обозначения
## Данные в задачах обучения по прецедентам
- Задача обучение с учителем
	- $X - \text{множество объектов}$
	- $Y - \text{множество ответов}$
	- $y: X \rightarrow Y \text{неизвестная зависимость (target function)}$
	- Дано: 
		- ${x_{1}, \dots, x_{l}} \in X - \text{обучающая выборка (training sample)}$
		- $y_{i} = y(x_{i}), i = 1, \dots, l$
	- Найти алгоритм $a: X \rightarrow Y$, решающую функцию (desicion funciton), приближающую y на всем множестве
- Весь курс построен на данной задаче:
	- Как задаются объекты и каким могут быть ответы
	- Каким образом мы строим a приближающую y 
	- Как строить функцию a
### Как задаются объекты. Признаковое описание
- $f_{j}: X \rightarrow D_{j}, j = 1, \dots, n \text{признаки объектов (features)}$
