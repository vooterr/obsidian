# IELTS High-Scoring Vocabulary: Top 100 Words

Этот список содержит 100 ключевых слов, необходимых для достижения высокого балла (7.0+) в секциях **Writing** и **Speaking**. Слова разбиты по категориям для эффективного применения в академическом контексте.

---

## 1. Cohesion & Coherence (Слова-связки)
*Используются для логического перехода между аргументами.*

| Слово | Перевод | Контекст использования |
| :--- | :--- | :--- |
| **Furthermore** | Более того | Добавление веского аргумента |
| **Moreover** | Кроме того | Расширение текущей мысли |
| **Nevertheless** | Тем не менее | Контраст с предыдущим утверждением |
| **Consequently** | Следовательно | Описание прямого результата |
| **Alternatively** | В качестве альтернативы | Предложение другого варианта |
| **Conversely** | Наоборот / С другой стороны | Прямое противопоставление |
| **In contrast** | В отличие от | Сравнение двух разных объектов |
| **Subsequently** | Впоследствии | Хронологическая последовательность |
| **In addition** | В дополнение | Добавление информации |
| **Therefore** | Поэтому | Вывод из сказанного |
| **Albeit** | Хотя и | Уступка (short for "although it be") |
| **Nonetheless** | Тем не менее | Синоним "nevertheless" |
| **Whereas** | В то время как | Сравнение двух фактов в одном предложении |
| **Primarily** | В первую очередь | Выделение главной причины |
| **Notably** | Примечательно | Акцент на важном факте |
| **Specifically** | В частности | Уточнение информации |
| **Ultimately** | В конечном счете | Финальный вывод или итог |
| **Hence** | Отсюда / Следовательно | Логическое следствие |
| **Despite** | Несмотря на | Противопоставление обстоятельств |
| **To illustrate** | Для иллюстрации | Введение примера |

## 2. Evaluation & Analysis (Академическая оценка)
*Для выражения мнения и критического анализа проблем.*

1.  **Significant/major** — Значительный
2.  **Substantial** — Существенный (о количестве/объеме)
3.  **Detrimental** — Пагубный / Вредный
4.  **Beneficial** — Выгодный / Полезный
5.  **Crucial** — Решающий / Ключевой
6.  **Essential** — Необходимый
7.  **Adequate** — Достаточный
8.  **Inadequate** — Недостаточный
9.  **Profound** — Глубокий (о влиянии или изменениях)
10. **Ambiguous** — Двусмысленный
11. **Feasible** — Осуществимый / Реальный
12. **Inevitably** — Неизбежно
13. **Prevalent/widespread** — Широко распространенный
14. **Viable** — Жизнеспособный
15. **Exacerbate** — Усугублять (проблему)
16. **Mitigate** — Смягчать / Облегчать (последствия)
17. **Augment** — Увеличивать / Дополнять
18. **Diminish** — Уменьшать
19. **Advocate** — Выступать за / Поддерживать (идею)
20. **Controversial** — Спорный

## 3. Action & Processes (Глаголы процессов)
*Для описания действий правительства, компаний или индивидов.*

* **Implement** — Внедрять (план, закон)
* **Establish** — Устанавливать / Основывать
* **Eradicate** — Искоренять / Уничтожать полностью
* **Facilitate** — Способствовать / Облегчать (процесс)
* **Utilize** — Использовать (с пользой)
* **Acquire** — Приобретать (навык, знание)
* **Obtain** — Получать (формально)
* **Maintain** — Поддерживать (уровень, состояние)
* **Accelerate** — Ускорять
* **Transform** — Преобразовывать
* **Encouraging** — Обнадеживающий
* **Hindering** — Препятствующий
* **Contribute** — Вносить вклад
* **Address** — Решать / Рассматривать (проблему)
* **Adjust** — Приспосабливаться / Корректировать
* **Encounter** — Сталкиваться (с трудностями)
* **Execute** — Выполнять / Исполнять
* **Modify** — Модифицировать / Изменять
* **Predict** — Предсказывать
* **Verify** — Проверять / Подтверждать
* **Outweight** — Перевешивать

5.  **Gradually** — Постепенно

9.  **Predominantly** — Преимущественно
10. **Scarcity** — Дефицит / Нехватка
11. **Abundance** — Изобилие
12. **Proportion** — Пропорция / Доля
13. **Majority** — Большинство
14. **Minority** — Меньшинство

18. **Relative** — Относительный
