# BFS(Breadth-First Search)

## 📝 Основное содержание

### Алгоритм
- Алгоритм обхода графа в ширину, исследующий слой за слоем, в порядке возрастания расстояния от источника
- Алгоритм основан на *очередях*, поэтому узлы обходятся по удалению вершин

### Для чего используется
- *Находит расстояние в ребрах до вершины*
- *Проверяет двудольность графа*
- *Обход по слоям дерева*
>BFS корректен **только** для невзвешенных графов (или равных весов)
   Каждая вершина попадает в очередь **ровно один раз** — отсюда O(n+m)
   0-1 BFS — не очевидная но мощная модификация, знай её
   Проверка двудольности через BFS: граф двудольный ↔ нет нечётных циклов ↔ все соседи `u` имеют `dist % 2 ≠ dist[u] % 2`
### Асимптотика BFS
- Память: $O(n + m)$ или $O(n^{2})$ в зависимости от реализации
- Время: $O(n + m)$
	- Проходимся по всем вершинам, из каждой вершины смотрим ребра
```C++
vector<int> bfs(int s, int n, vector<vector<int>>& adj) {
    vector<int> dist(n, -1);
    dist[s] = 0;
    queue<int> q;
    q.push(s);

    while (!q.empty()) {
        int u = q.front(); q.pop();
        for (int v : adj[u]) {
            if (dist[v] == -1) {
                dist[v] = dist[u] + 1;
                q.push(v);
            }
        }
    }
    return dist;
}
```
- В случае необходимости восстановления пути, в операцию после условия добавляется инициализация родителя узла
