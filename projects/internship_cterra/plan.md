# Экспорт модели в C++
- [ ] Выбрать один из вариантов ниже
	- - **ONNX** (универсально):
    
    - sklearn → `skl2onnx` (или `onnxruntime-training`/`onnxmltools`),
        
	    - LightGBM/XGBoost → экспорт в ONNX (или через `microsoft/hummingbird-ml` → ONNX).
        
	    - Плюсы: есть быстрый `onnxruntime` под C/C++.
        
	- **Treelite** (для деревьев XGB/LGBM): конвертирует модель в оптимизированную C-библиотеку.
    
	- **m2cgen** (склонен к простым моделям): генерирует C-код из модели sklearn (ограничения по сложным пайплайнам).
