## 4. Операции

### OP-1: Daily Review
Триггер: "ревью", "вечернее ревью", "что я сделал".
```
1. Читай daily/YYYY-MM-DD.md
2. Читай Google Calendar за день
3. Сравни план vs факт
4. Запиши в _claude/review/YYYY-MM-DD.md
5. Обнови _claude/log.md
6. Скорректируй Calendar завтра если нужно
```
Не пиши ревью если нет ни дневника, ни учебных событий. Только строка в log.

### OP-2: Schedule Update
Триггер: "запланируй", "добавь в календарь", "перенеси".
```
1. Calendar: list_events(week)
2. Проверь конфликты с жёсткими блоками
3. Создай/обнови событие
4. Логируй в _claude/log.md
```

### OP-3: Ingest
Триггер: "добавь конспект", "законспектируй", "я прочитал X".
```
1. Получи материал
2. Извлеки концепты, формулы, примеры
3. Создай/обнови файл в resource/ (нужная подпапка)
4. Обнови _claude/index.md
5. Логируй в _claude/log.md
```

### OP-4: Weekly Review
Триггер: "недельное ревью", воскресенье.
```
1. Calendar: list_events(week)
2. Читай _claude/review/ за неделю
3. Считай часы по приоритетам A/B/C
4. Обнови _claude/roadmap.md
5. Скорректируй следующую неделю в Calendar
```

### OP-5: Lint
Триггер: "lint", "проверь вики".
- Файлы в корне не в своих папках
- Страницы без входящих ссылок
- Противоречия между конспектами
- Пустые/заброшенные файлы
